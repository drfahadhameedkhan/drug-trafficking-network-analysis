# Data Sources

This folder documents data sources for the network analysis.
The network is constructed programmatically in the notebook.

## Empirical Data Sources (for future extension)
- UNODC World Drug Report: https://www.unodc.org
- UNODC Seizure Data: https://dataunodc.un.org/
- Global Organized Crime Index: https://ocindex.net/

## Node Attributes
- `role` : Supplier / Transshipment / Distributor / Market
- `region` : South Asia
- `weight` : Relative size/capacity (2–10)

## Edge Attributes
- `weight` : Trafficking flow strength (1–5)
